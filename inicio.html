Estructura básica de html5 (etiquetas, atributos y valores)

Objetivo General

     El proposito de este proyecto es crear un recurso de aprendizaje sobre la Estructura básica de html5 (etiquetas, atributos y valores) que permita dinamizar las estrategias de aprendizaje, al terminar el estudiante debe estar e la capacidad, con todo el conocimiento adquirido de elaborar un sitio web, aplicando todos los conceptos de HTML5.

Objetivos Especificos

  • Aprender a usar las diferentes etiquetas.
  • Identificar los diferentes tipos de atributos para una mejor aplicacion a la hora de diseñar un sitio web.
  • Conocer las sintaxis de la escturuta basica de html5.
